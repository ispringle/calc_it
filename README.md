# calc_it
A calculator in Rust

## Usage
`./calc_it [FLAGS] <a> <OPERATOR> <b>`

### Flags
- `v`/`verbose`

### Supported operators
- Additioan `+`
- Subtraction `-`
- Multiplication `x`
- Division `/`
- Modulo `%`
